📍 Here you can draw the searching area on the map, or geolocate.

If you geolocate, draw a point or a line, it will automatically create an area of 100 meters around this one (buffer size customizable). If you draw a circle or polygon, it will search in this area.
Don't search in a too big area because it would be too long to load, even if there is a default limit to 3000 species observations maximum requested in the GBIF API.
